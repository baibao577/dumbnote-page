# Keyboard Shortcuts

Complete reference of all keyboard shortcuts in DumbNote. 

> **Note**: `Cmd` is for macOS, `Ctrl` is for Windows/Linux

## Essential Shortcuts

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Cmd/Ctrl+N` | New Note | Create a new note |
| `Cmd/Ctrl+S` | Save | Save current note (manual save) |
| `Cmd/Ctrl+F` | Find | Context-aware search |
| `Cmd/Ctrl+Shift+V` | Toggle Preview | Switch between edit and preview |
| `Cmd/Ctrl+B` or `Cmd/Ctrl+\` | Toggle Sidebar | Show/hide sidebar |
| `Escape` | Close/Cancel | Close dialogs, reset states |

## Pro Tips

### Combination Shortcuts
Some shortcuts work in sequence:
- `Cmd/Ctrl+K, Cmd/Ctrl+0` - Fold all (press K, release, then press 0)
- `Cmd/Ctrl+K, Cmd/Ctrl+D` - Skip occurrence in multi-select

### Efficiency Tips
1. **Quick formatting**: Select text, then apply format
2. **Multi-cursor**: Edit multiple locations simultaneously
3. **Line operations**: Quickly reorganize without copy/paste
4. **Folding**: Navigate large documents efficiently

### Customization
While DumbNote doesn't support custom keyboard shortcuts, you can:
- Use system-level remapping tools
- Create text expansions for frequently used snippets
- Use the built-in text shortcuts (@date, ::, etc.)
